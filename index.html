Projects

My three favorite projects are listed below. See my GitHub for a complete list of my work.

Polio Vaccination Coverage

Story | Code
For this project I wanted to know how Polio vaccine coverage has changed since it was first introduced in 1980. I gathered data from the World Health Organization (WHO) website and conducted exploratory Analysis to visualize the vaccination coverage and how it has effected the spread of the disease over the past 4 decades. The dashboard visualizes coverage by yer, country and segments. The data was cleaned using Python library Pandas queried in sqlite.

Mission to Mars

Code
For this project I built a webscrapper in Python and created a local website where specific information about Mars based on the latest information on the NASA website, was displayed. This project used my knowledge of Python, MongoDB, HTML and CSS.

Emotional Recognition using Neural Networks

Code
With the latest advancements in image recognition technologies, I was interested in learning about how neural networks could be trained on images and how these trained models could be used to predict emotions on faces in images. I used many open source libraries like keras, scikit learn and OPEN CV2 and images extracted from an online resource to train my Neural network. This process took 12 hours and the resulting model allowed for the prediction of images with four facial expressions
